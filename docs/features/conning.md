# Conning Display Features

This document tracks the detailed features for the Conning Display in the Ship Simulator. Features are based on real-world conning displays (e.g., Furuno) and project goals. Checked items are implemented or present in code/UI; unchecked are pending or unclear.

## Core Conning Display
- [ ] Central ship schematic with heading indicator
- [ ] Rudder angle indicator (actual and commanded)
- [ ] Propeller indicators (azimuth, pitch, RPM, direction)
- [ ] Engine telegraph status (ahead/astern, setpoint, feedback)
- [ ] Bow thruster and stern thruster indicators
- [ ] Rate of turn indicator
- [ ] Wind direction and speed (true and relative)
- [ ] Speed log (through water and over ground)
- [ ] Position info (latitude, longitude, time, GPS status)
- [ ] Depth under keel
- [ ] Pitch and roll gauges
- [ ] Heading (gyro and magnetic)
- [ ] Course over ground (COG) and speed over ground (SOG)
- [ ] Alarms and warnings (steering, propulsion, navigation)
- [ ] Navigation mode/status (manual/auto, track control, etc.)
- [ ] Docking aids (distance to berth, fender lines)
- [ ] Clear, at-a-glance layout for maneuvering and navigation
