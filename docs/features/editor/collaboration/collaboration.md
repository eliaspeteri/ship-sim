# Collaboration Model

## Purpose

Support multi-user editing with presence and conflict handling.

## Scope

- Presence and cursors
- Locking and edit ownership
- Conflict resolution

## Out of Scope

- Review approval
- Moderation

## Implementation Checklist

- [ ] Presence list in editor shell
- [ ] Feature-level lock/claim
- [ ] Soft conflict warnings
- [ ] Per-user visibility state
- [ ] Audit trail for changes
