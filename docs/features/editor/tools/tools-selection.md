# Selection & Manipulation Tools

## Purpose

Select and transform features reliably across all layers.

## Scope

- Click and lasso selection
- Multi-select
- Move/rotate/scale
- Duplication
- Soft locks and warnings

## Out of Scope

- Creation tools
- Inspector forms
