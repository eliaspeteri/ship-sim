# Select Tool

## Purpose

Select and inspect existing features in the viewport.

## Scope

- Click select
- Multi-select
- Hover highlight
- Selection clearing

## Out of Scope

- Editing geometry
- Property editing (handled in inspector)

## Implementation Checklist

- [ ] Click to select feature under cursor
- [ ] Shift-click to add/remove from selection
- [ ] Esc clears selection
- [ ] Hover outline when tool active
- [ ] Selection state syncs with inspector
