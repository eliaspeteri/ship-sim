# Draw Tool

## Purpose

Create polyline or polygon features by drawing vertices in the viewport.

## Scope

- Vertex placement
- Line/polygon completion
- Undo last vertex

## Out of Scope

- Freeform painting

## Implementation Checklist

- [ ] Click to add vertex
- [ ] Double-click or Enter to finish
- [ ] Backspace removes last vertex
- [ ] Polygon auto-closes when appropriate
- [ ] Temporary preview line while drawing
