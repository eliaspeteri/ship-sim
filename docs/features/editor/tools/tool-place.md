# Place Tool

## Purpose

Place new point or instance features into the current layer.

## Scope

- Single-click placement
- Placement preview
- Cancel placement

## Out of Scope

- Line/polygon drawing

## Implementation Checklist

- [ ] Preview ghost at cursor
- [ ] Click to place feature
- [ ] Respect active layer type/geometry
- [ ] Snap to grid/feature when enabled
- [ ] Esc cancels placement
