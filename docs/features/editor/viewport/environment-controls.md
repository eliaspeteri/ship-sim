# Editor Environment Controls

## Purpose

Allow authors to preview overlays under different lighting and weather.

## Scope

- Lighting presets
- Time of day
- Weather presets
- Preview toggles

## Out of Scope

- Full simulation state changes

## Controls

- Sun angle / time slider
- Fog / visibility presets
- Weather presets (clear, overcast, rain)
- Toggle water/atmosphere effects
